Time and Space Complexity

Time Complexity is understood as the time taken by a program to run and complete the guven task. As a programmer,your goal should be reduce the Time Complexity of the code by optimizing it and make it more efficient.

Space Complexity is a related to the amout of memory occupied by the program.For exampl, if a program takes an array of size n as input,the Space Complexity of the program become O(n)
